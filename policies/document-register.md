# Document Register

This register is the authoritative index of all controlled documents within the Helge Heupel Group Integrated Management System (IMS), covering both the ISMS (ISO 27001) and AIMS (ISO 42001). It is maintained as a living document and updated each time a new controlled document is created, revised, or retired.

**Maintained by:** CISO Agent (with human CISO oversight)
**Update trigger:** Creation, revision, or retirement of any controlled document
**Review frequency:** Updated continuously; formally reviewed semi-annually
